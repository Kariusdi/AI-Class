จาก Data set ที่ได้ พบว่ารูปแบบของ Data set คือ
มีการแบ่งชุดข้อมูลเป็นแนวเฉียงจากค่า Cov แต่แต่ละกลุ่มข้อมูลเกาะกลุ่มกันเป็น 2 จุด

โดยเริ่มแรกเราควร Normalize data ให้อยู่ในช่วงเดียวกันก่อนเพื่อลดปัญหา Bias ของ Model และกำหนด Lable (y) ให้แต่ละชุดข้อมูลเพื่อไว้สำหรับเช็คประสิทธิภาพได้

จากนั้นก็ทำการออกแบบ Model โดยเรานั้นสามารถใช้ Layer และจำนวน Node ที่ไม่มากในการสร้างได้เนื่องจากเราไม่ต้องการเส้นที่พับกันหลายชั้นเพื่อให้ได้เส้นแบ่งที่โค้ง (More layers) ขนาดนั้นเช่นกัน เพราะ แค่สามารถแบ่งเส้นออกกลุ่มใดกลุ่มนึงให้ได้ (ครบทั้ง 2 จุดที่กลุ่มข้อมูลอยู่)และอีกกลุ่มนึงอาจจะได้แค่กลุ่มเดียว (ครบจุดเดียว จุดที่กลุ่มข้อมูลอยู่) ก็สามารถแยก Class ได้ประสิทธิภาพสูงกว่า 70% แล้ว ผมจึงเลือกการออกแบบ Layer แค่ 4 Layer แต่ใช้ Node น้อย โดยแบ่งเป็นชั้นละ 8, 4, 4, 1 Nodes เท่านั้น และกำหนดให้ใช้ Activation Function ใน Layer แรกเป็น Tanh เพราะอยากได้ความ Slope ในการแบ่งข้อมูลเนื่องจาก Dataset นี้ข้อมูลมีความเฉียง และใช้เป็น Relu เนื่องจาก
สามารถแยก Data set แบบนี้ได้ดีเช่นเดิม รวมถึงปิดด้วย sigmoid

อีกทั้งในตอนสร้าง Model ผมได้มีการกำหนดการแบ่ง test size ให้เป็น 0.5 เพื่อป้องกันปัญหา Over-fitting และ Batch size เป็นทีละ 10 จากข้อมูล 100 และรอบการเทรน (epochs) เป็น 70 เพื่อเพิ่มความละเอียดในการเทรนใน Layer ที่น้อยและลดปัญหา Under-fitting เช่นกัน
